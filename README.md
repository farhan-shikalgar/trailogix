
# Trailogix

A website for booking trailer


## Tech Stack

**Client:** Nextjs,React

**Server:** MongoDb,Node


## Run Locally

Clone the project

```bash
  git clone https://github.com/farhan-shikalgar/trailogix.git
```

Go to the project directory

```bash
  cd trailogix
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```

Use this Credentials if not want to register(Admin)

```bash
  User-test
  pass-123456
```
## Features

- Admin panel
- To book trailer
- To view trailers that are booked
- responsive
- authentication
- login and register
